## Usage
To use printf you will need the basic syntax.
```
#include "main.h"

int main(void)
{
    printf("Hello Word!\n");
    return(0);
}
```

## Format Specifiers

```%d``` Prints an interger 

```%i``` Prints an interger

```%c``` Prints a character 

```%s``` Prints a string 

```%%``` Prints a percent sign
